# MK World Lounge Score Tracker

## 目的
マリオカートワールドのラウンジ（1模擬12レース）で、順位・ポイントを記録し、統計を可視化するためのブラウザアプリです。

## 出来ること
- 模擬中のレース結果（順位/総合順位）を入力
- 配点ルールに基づいてポイントを自動計算
- 模擬平均順位・累計ポイントの表示
- 12人/24人別の統計（平均順位・勝率など）
- 模擬履歴の保存・削除
- データのエクスポート/インポート

## コース名の入力方法
- 例: `マリオブラザーズサーキット` などの正式名称
- 例: `まりぶら` など `aliases` に登録した略称
- 道コースを入力する場合は先頭に `道` を付ける
  - 例: `道マリオブラザーズサーキット`
  - 例: `道まりぶら`

※ コース入力は任意です。未入力でもレース追加できます。

## コース名略称の追加方法
`course.json` を編集します。

- 例: `aliases` に略称を追加

```
{
  "id": "1",
  "name": "マリオブラザーズサーキット",
  "aliases": ["まりぶら", "マリブラ"]
}
```
↓↓↓

```
{
  "id": "1",
  "name": "マリオブラザーズサーキット",
  "aliases": ["まりぶら", "マリブラ","マリブサ","まりぶさ"]
}
```

※ `index.html` 内に埋め込みのコースデータがある場合、`course.json` と同じ内容に更新してください。

## 注意事項
- 本ソフトウェアはMKCentralの許諾を受けたものではありません。
- 利用は自己責任で行ってください。
- コース名は日本語のみ対応です。
- コース名の略称は独断と偏見で入れています。
- 逆走マリブサ、クリボートロフィー、水トロフィー等の特殊コースは追って対応します。
